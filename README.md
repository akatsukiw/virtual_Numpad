# virtual_Numpad
无焦点悬浮数字虚拟软键盘。在只能一个手控制鼠标和输入数字的时候会很有用。


---

## ✨ 功能说明

### 🛠️ 调整大小

按住窗口右下角的 **"◢"** 符号拖动即可改变键盘尺寸，按钮会根据窗口大小自动伸缩适配，满足不同屏幕分辨率的需求。

### 📌 置顶开关

点击标题栏右侧的 **📌** 按钮，可一键切换窗口是否始终显示在最表层，确保在录入数据时不会被 Excel 窗口遮挡。

### 🎚️ 透明度调节

点击 **⚙️ 齿轮图标** 展开高级设置面板，通过拖动滑动条可实时调整窗口透明度。

 **权限管理**：程序内置管理员权限申请，确保在 UAC 开启的状态下依然能向高权限软件发送指令。

## 📥 下载与运行

如果你只想直接使用，请前往 **[Releases 页面]** 下载：

1. **单文件版 (`Numpad_SingleFile.exe`)**：
* 由 Nuitka 编译，独立运行。


2. **文件夹版 (`Numpad_Stable_Folder.zip`)**：
* 由 PyInstaller 打包，包含完整依赖库。如果单文件版在你的老旧电脑上无法打开，请优先使用此版本。

## 👩‍💻 开发者模式 (自己运行)

如果你想在 Python 环境下运行或二次开发：

### 1️⃣ 安装依赖

```bash
pip install keyboard

```

### 2️⃣ 下载代码

将源代码下载 `numpad.py`。

### 3️⃣ 启动项目

```bash
python numpad.py

```

## ⚠️ 使用须知

* **管理员权限**：由于需要全局监听并发送按键，程序**必须以管理员身份运行**。
* **输入法建议**：若在某些特定软件下无法录入，请尝试将输入法切换至 **英文 (美国)** 状态。
* **报毒说明**：由于按键模拟逻辑会被部分杀毒软件判定为“风险行为”，若被拦截请添加至白名单。
